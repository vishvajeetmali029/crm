# 🚀 CRM-Based Course Management System (MERN Stack)

A full-stack CRM application to manage the complete student lifecycle — from enquiry to admission, batch allocation, and payment tracking.

---

## 📌 Features

### 🔐 Authentication

- JWT-based login system
- Role-based access (Admin, Counselor, Trainer, Student)

### 📞 Enquiry Management

- Add and track enquiries
- Follow-up tracking
- Convert enquiry → student

### 🎓 Student Management

- Student registration
- Lifecycle tracking (Active, Completed, Dropped)
- Batch assignment

### 📚 Course & Batch Management

- Create and manage courses
- Batch scheduling and capacity handling
- Assign students to batches

### 💰 Payment System

- Installment-based payment tracking
- Auto calculation of paid & due amount
- Prevent overpayment
- Payment history

### 📊 Dashboard Analytics

- Total students
- Total enquiries
- Revenue tracking
- Pending dues

---

## 🧠 Tech Stack

### Frontend
